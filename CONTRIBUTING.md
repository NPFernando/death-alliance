# Contributing to Death Alliance

Thank you for helping keep this fictional / ARG / roleplay archive concept alive.

This project exists as a safe creative storytelling and public-archive design experiment. Contributions are welcome when they preserve the core safety boundary:

- Fictional, anonymized, roleplay, ARG, lore, or worldbuilding content only.
- No real-world accusations against identifiable real people.
- No doxxing, harassment, threats, revenge content, private personal data, or unsafe metadata.
- No browser-persistent hidden keys or identity secrets.
- No malware-like spreading, hidden persistence, takedown bypass, or evasion behavior.

## How to contribute

1. Fork the repository.
2. Create a feature branch.
3. Run validation locally:

```bash
npm install
npm run validate
```

4. Open a pull request with:
   - What changed.
   - How you validated it.
   - Screenshots or short clips for UI changes when useful.
   - Confirmation that the change keeps the project fictional/safe.

## Attribution request

The MIT License allows forks, self-hosted instances, modifications, and redistribution.
